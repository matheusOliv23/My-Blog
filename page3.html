Método: Map()

Métodos mais utilizados

Neste tópico, iremos falar sobre um dos métodos mais utilizados no Javascript.

Map: .map()

A função desse método é criar um novo array com os valores modificados.
No exemplo acima, criamos um array de números. Criamos uma outra variável chamada 'double' cuja função é dobrar os números do array 'números'. Perceba como foi fácil resolver esse problema utilizando o map, com uma sintaxe bem simples e fácil de entender.
O .map() possui três argumentos. São eles: o valor do elemento, o índice do elemento e o array do elemento.

Conclusão

Aprendemos como utilizar o map para percorrer valores de um array e modifica-los de uma maneira bem simples. Mais informações poderão ser consultadas na sua documentação. O link te redicionará à uma nova página sem sair da atual.
