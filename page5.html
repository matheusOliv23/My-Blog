Método: forEach()

.forEach()

Caso precise executar algum código para iterar elementos, é mais fácil executar um .forEach() do que fazer um for ou while.
No exemplo acima, criamos um array de números. Veja como foi simples utilizar o método .forEach para percorrer cada elemento do array e dobrar o seu valor.
Pode facilitar muito a sua vida utilizar ela ao invés do for.
O .forEach() possui três parâmetros. São eles: o valor do elemento, o índice do elemento e o array que está sendo percorrido.

Conclusão

Vimos como utilizar o forEach para percorrer os valores de um array de maneira descomplicada. Utilize a documentação caso precise de mais informações. O link te redicionará à uma nova página sem sair da atual.
